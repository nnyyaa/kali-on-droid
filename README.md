这是我的第一个作品

（我是菜鸡，只会搞一些基础的东西qaq）

食用方法：


安装termux，然后换国内源，之后
```
cd ~
pkg install git -y
git clone https://hub.fastgit.xyz/wspbh/kali-on-phone.git
cd kali-on-phone
chmod +x install.sh
```  
然后
```
./install.sh
```
